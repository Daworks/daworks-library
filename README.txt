작은 도서관용 도서관리 플러그인

== Description ==

작은 도서관을 위한 도서관리 플러그인입니다. 도서목록을 표시하고, 보유 도서 검색, 등록, 삭제, 업데이트 가능합니다.

== Installation ==

깃허브에서 플러그인을 다운 받은 후 플러그인 디렉토리에 복사하고 플러그인을 활성화하세요.
저장소 : https://github.com/Daworks/daworks-library.git

1. 숏코드로 표시하세요.
1. 도서목록 표시 숏코드 : [daworks_church_library]
1. 최근 등록 도서목록 숏코드 : [daworks_church_library_latest nums=5]
